# 总需求
给我写一个python CLI 程序，实现一个本地的模糊文档查询,
使用 opensearch 将 大量的.md文件做成数据【我hexo写的笔记】
使用 opensearch 来增删改查，查询数据

# 项目目录要求如下：
data/
	1.txt
	2.txt
	....
tmp/
	1.json
	...
src/
	xxx.py
	....
main.py


# 交互上：
按键1: 将数据库建立倒排索引
按键2： 查询文章(只返回文章的出处) 
按键3： 全部重新建立倒排索引
按键exit: 退出交互


# 数据
data下的数据为中英文混合的文章，字数大概在10W字以内

## 运行说明

### 环境准备
- Python 3.12+
- 安装依赖：
  ```bash
  uv sync
  ```

### 使用
```bash
python main.py
```

交互菜单：
- 按键1：建立/增量更新倒排索引（仅对新增或修改的文件重新索引）
- 按键2：查询文章（仅显示文章文件路径，按相关度排序）
- 按键3：全量重建倒排索引（忽略缓存，重新索引全部文件）
- 按键exit：退出
